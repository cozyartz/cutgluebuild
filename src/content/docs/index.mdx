## Platform Overview

CutGlueBuild combines several key technologies:

- **Frontend**: Astro 4.x with React components and TypeScript
- **Styling**: Tailwind CSS with dark mode support
- **Backend**: Supabase for authentication, database, and edge functions
- **AI Integration**: OpenAI GPT-4 for SVG generation and project ideas
- **Payments**: Stripe for subscription management
- **Content**: MDX for blog posts and documentation
